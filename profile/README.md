# ✨ Key Features

- 🏛 **Nation Profile Manager:** Save and switch between country setups  
- 🧩 **Mod Organizer:** Manage gameplay, economy, and overhaul mods  
- 💾 **Auto Save Backup:** Protect campaign progress automatically  
- 🛡 **Safe Configuration Layer:** No modification of core game files  
- 🌍 **Cross-Platform Support:** Windows, macOS, Linux compatible  
- ⚡ **Performance Optimization:** Improve late-game simulation stability  
- 📊 **Economic Scenario Switcher:** Quickly load different economic builds
